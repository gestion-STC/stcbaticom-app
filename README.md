# STC Bâtiment — Logiciel commercial (module Call)

CRM de prospection STC (appels Ringover, pipeline, emails Resend).
Base : Supabase STCBATICOM. Interface publiée automatiquement :
https://gestion-stc.github.io/stcbaticom-app/

⚠️ Dépôt PUBLIC — aucun secret ici. `.env.local` ne contient que des clés publiques.
Développement : `npm install` puis `npm run dev` · Build : `npm run build` · Tests : `npm run test`
